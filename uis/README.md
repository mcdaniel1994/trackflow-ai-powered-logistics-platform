# `uis/`

Forward-looking UI workspace for TrackFlow's Next.js + TypeScript interfaces.

## Purpose

- `uis/website/` - public website moving forward from Engagement 4
- `uis/backoffice/` - internal company tools moving forward from Engagement 4, including the Talent Pipeline Tracker at `/talent` (migrated from its standalone app in June 2026)

`uis/` is now the sole UI home. The old standalone apps (`apps/marketing-site/`, `apps/talent-pipeline-tracker/`) were retired in June 2026 after their surfaces were verified in `uis/`; see `docs/archive/marketing-site-retirement.md` and `docs/archive/talent-pipeline-tracker-retirement.md`.
