# `uis/`

Forward-looking UI workspace for TrackFlow's Next.js + TypeScript interfaces.

## Purpose

- `uis/website/` - public website moving forward from Engagement 4
- `uis/backoffice/` - internal company tools moving forward from Engagement 4

The delivered historical apps in `apps/` remain in place. Engagement 4 does not migrate or remove `apps/marketing-site/` or `apps/talent-pipeline-tracker/`.
