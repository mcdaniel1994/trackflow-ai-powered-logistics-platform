# `skills/code-review/`

Scaffolding folder for reusable code review skill assets.

## Purpose

This folder is reserved for code review workflows, prompts, and supporting assets used by agents during repository and pull request reviews.

## Current State

The folder is currently a placeholder (`.gitkeep`) and does not yet define an active skill.

## When Implementing

Add the following as needed:

- `SKILL.md` with clear use criteria and workflow steps
- `resources/` for reference material and checklists
- `scripts/` for helper automation used by the skill
- `examples/` for sample inputs and outputs

Keep the structure and naming aligned with `skills/README.md` and `skills/_template/`.
