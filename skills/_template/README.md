# `skills/_template/`

Starter pattern for creating a new agent skill.

## Purpose

Provides the minimal scaffold (`SKILL.md`, `examples/`, `resources/`, `scripts/`) so new skills under `skills/` start from a shared standard.

## How to Use

1. Copy this folder, rename it to the new skill's name.
2. Fill in `SKILL.md` (purpose, when-to-use, inputs, outputs, example).
3. Add reference material to `resources/`, sample I/O to `examples/`, helper scripts to `scripts/`.

`SKILL.md` itself is the authoritative template — keep it in sync with conventions used by sibling skills.
