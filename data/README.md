# `data/`

Data engineering assets for the TrackFlow platform.

| Folder | Purpose |
|---|---|
| `raw/` | Source data: exports, dumps, sample files, untransformed datasets |
| `process/` | Cleaned and normalized data produced by pipelines |
| `pipelines/` | ETL/ELT logic: ingestion, transformation, loading |
| `eval/` | AI evaluation datasets for testing agent and model outputs |

This folder is scaffolding. Real data assets land in **Engagement 6** (data pipelines & telemetry).
