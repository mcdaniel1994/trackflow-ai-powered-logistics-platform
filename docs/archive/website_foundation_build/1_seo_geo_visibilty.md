As we plan this web app, ensure every decision accounts for:
- Semantic HTML (meaningful tags, logical structure)
- Web Accessibility (WCAG 2.1 AA: ARIA, keyboard nav, contrast, alt text)
- SEO (meta tags, canonical URLs, crawlability, page speed < 3s)
- GEO (answer-first structure, cite sources, include statistics, clear authorship and entity definitions)
- E-E-A-T (demonstrate experience, expertise, authority, and trust in content and structure decisions)
- Schema.org JSON-LD (WebSite, WebPage, Article, FAQPage, or SoftwareApplication as appropriate)
- Code quality (no bugs, clean separation of concerns, maintainable patterns)

Flag any architectural decision that would compromise these. Keep recommendations practical and specific to what we're building.
