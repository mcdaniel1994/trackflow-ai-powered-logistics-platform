This folder is for aggregate-only evidence, not source data. Images placed here must not include raw incident rows, customer email addresses, copied CSV contents, or offending field values from the official local dataset.

- `backoffice-analysis-results.png` - backoffice `/incidents` aggregate analysis view.
- `cli-analysis-results.png` - terminal output from the incident analysis CLI.
